# Privacy Policy — T4H Widget Dock

**Last updated: 27 March 2026**

## Data collection

T4H Widget Dock does not collect, store, or transmit any personal data to Tech 4 Humanity servers.

## What the extension accesses

- **Your Supabase API key** — stored locally in Chrome's storage API, never transmitted to T4H
- **Your Supabase database** — queried directly from your browser using your own credentials
- **Active tab URL** — read-only, used for context display only, never stored

## Data storage

All configuration (API key, preferences) is stored in `chrome.storage.local` — local to your browser only.

## Third parties

This extension makes direct requests to:
- Your Supabase instance (your infrastructure)
- T4H product domains (for widget snippet delivery only)

No analytics, advertising, or tracking services are used.
